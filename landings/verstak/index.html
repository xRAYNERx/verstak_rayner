Как работает

От открытия проекта до production — четыре шага

Открой проект

Verstak видит структуру, индексирует код, читает CLAUDE.md / AGENTS.md для контекста и правил.

Выбери модель

Клик в композере — выбор провайдера и модели. Подсчёт стоимости токенов в реальном времени.

Куй код

Plan mode для сложных задач, Autopilot для рутины, multi-file diff с подтверждением каждой правки.

Контролируй

Undo-стек, kill switch (Shift+Esc), secret-scanner, path policy. Ничего не уходит без твоего согласия.

FAQ

Частые вопросы

Чем Verstak отличается от Cursor?
Тремя вещами: (1) любая модель — не привязан к Anthropic; (2) использует твои CLI-подписки вместо токенов API; (3) сделан под РФ — GigaChat / YandexGPT нативно, оплата картой РФ, 1С-коннекторы, юрлицо в России.
А я уже плачу за Claude Pro / ChatGPT Plus — это пригодится?
Да. Verstak умеет работать через подписочные CLI (Claude Code, Codex, Gemini, Grok). Ставишь CLI один раз, логинишься, и Verstak использует твою подписку — не берёт ни рубля за токены. Если CLI ещё не вышел для нужного провайдера, Verstak подключается через API по твоему ключу.
Безопасны ли мои API-ключи и код?
Ключи хранятся в Electron safeStorage (системный keychain). Secret-scanner блокирует чтение .env, .ssh, *.key, credentials. Все output'ы инструментов проходят через регулярки, редактирующие API-ключи и приватные блоки как [REDACTED:type]. Код никуда не уходит, кроме выбранного тобой провайдера модели.
Какие операционные системы поддерживаются?
Windows 10/11, macOS 12+ (Intel и Apple Silicon), Linux (Ubuntu 22.04+, Fedora, Arch). Сборки — Electron, нативная производительность, ~150 МБ дистрибутив.
Нужен ли VPN для GigaChat / YandexGPT?
Нет. Verstak обращается к российским API напрямую с российских IP. Никакого VPN, никаких прокси. Это и есть одна из главных причин Russia-first позиционирования.
Поддерживается ли on-premise развёртывание?
Да, на Enterprise-тарифе. Можно поставить Verstak в закрытый контур, подключить локальные модели (GigaChat MLP, YandexGPT on-premise, llama.cpp), настроить аудит и SSO. Связывайтесь — sales@verstak.io.
Как происходит оплата для РФ?
Карта любого российского банка через ЮKassa / CloudPayments. Для юрлиц — счёт по реквизитам, акт, счёт-фактура с НДС. Никаких посредников, иностранных платёжных систем или криптовалют.
Что в roadmap?
Q2 2026: open beta, packaged installer, multi-chat per project. Q3: background agents, MCP-серверы, маркетплейс коннекторов, on-premise Enterprise. Q4: коллаборативные сессии, командные библиотеки промтов, биллинг по командам. Подписывайся на GitHub — релизы выходят каждые 2 недели.
